# Cinnabar Forge Meta

Configuration editor and version updater.

## Getting Started

### Installation

Install globally using npm:

```bash
npm install -g @cinnabar-forge/meta
```

This will make the `cinnabar-meta` command available in your terminal.

### Usage

```bash
cinnabar-meta
```
